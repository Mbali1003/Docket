
# Docket

## AI Productivity Assistant

### Project Description
Docket is an AI-powered productivity assistant designed to help users automate everyday workplace tasks. It improves productivity by providing tools for email generation, meeting summarization, task management, research assistance, and chatbot interaction.

## Features

- 📧 Email Generator
- 📝 Meeting Summarizer
- ✅ Task Planner
- 🔍 Research Assistant
- 💬 AI Chatbot

## Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your web browser.
3. Start using Docket.

## Responsible AI

Docket is designed to assist users with workplace tasks. Users should review AI-generated content before using it to ensure it is accurate and appropriate.

## Future Improvements

- Connect to the OpenAI API
